How to Do Zong Package

How to Activate a Zong Package

There are several ways to activate a Zong package. You can activate it by:

  • Dialing a USSD code.
  • Sending an SMS.
  • Visiting a Zong store.

To activate a Zong package by dialing a USSD code, follow these steps:

  1. Dial the USSD code for the package you want to activate. For example, to activate the Monthly Super Offer, you would dial *6464#.
  2. Follow the instructions on the screen.

To activate a Zong package by sending an SMS, follow these steps:

  1. Type the package code and send it to 6464. For example, to activate the Monthly Super Offer, you would type MS and send it to 6464.

To activate a Zong package by visiting a Zong store, follow these steps:

  1. Visit a Zong store near you.
  2. Tell the store attendant the package you want to activate.
  3. Pay the activation fee.

How to Check Your Zong Package Balance

You can check your Zong package balance by:

  • Dialing a USSD code.
  • Sending an SMS.
  • Visiting a Zong store.

To check your Zong package balance by dialing a USSD code, follow these steps:

  1. Dial *102#.
  2. The balance of your active packages will be displayed on the screen.

To check your Zong package balance by sending an SMS, follow these steps:Zong Prepaid Data Packages

  1. Type the keyword BAL and send it to 6464.
  2. You will receive an SMS with the balance of your active packages.

How to Unsubscribe from a Zong Package

You can unsubscribe from a Zong package by:

  • Dialing a USSD code.
  • Sending an SMS.
  • Visiting a Zong store.

To unsubscribe from a Zong package by dialing a USSD code, follow these steps:

  1. Dial the USSD code for the package you want to unsubscribe from. For example, to unsubscribe from the Monthly Super Offer, you would dial *6464# and press 5.

To unsubscribe from a Zong package by sending an SMS, follow these steps:

  1. Type the package code and send it to 6464. For example, to unsubscribe from the Monthly Super Offer, you would type MU and send it to 6464.

To unsubscribe from a Zong package by visiting a Zong store, follow these steps:

  1. Visit a Zong store near you.
  2. Tell the store attendant the package you want to unsubscribe from.
